第三章 镜像

3.1 获取镜像

sudo docker pull ubuntu:14.04

3.2 查看镜像

docker images

包含信息:

  • 仓库
  • 标签
  • ID
  • 创建时间
  • 大小

可以使用docker tag 添加一个新的标签

ID 相同的不同 TAG 其实是指向同一个镜像文件,只是别名不同

获取镜像详细信息: docker inspect
可以使用 -f 参数指定获取其中一项内容

3.3 搜寻镜像

docker search

默认搜索 DOcker hub 官方仓库

  • –automated=false 仅显示自动创建的镜像
  • –no-trunc=false 输出信息不截断显示
  • -s ,–stars=0 指定评价指定星级以上的镜像

3.4 删除镜像

docker rmi

-f 强制删除
